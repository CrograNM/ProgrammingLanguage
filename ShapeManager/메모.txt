

<Line이 잘 추가되었는가?>
- 수정할 필요가 없는 파일이 있었는가?
	: circle 등 기본 도형 파일들은 수정할 필요가 없었다.
	: 부모 클래스인 Shape 파일 등도 수정할 필요가 없었다.

- 수정할 부분이 있었다면 어느 파일의 어디였는가를 적어보라.
	: ShapeManager, Menu, main 에서 Line 도형을 추가하기 위해 수정이 필요했다.
	: ShapeManager 파일에서는 Line 도형을 인식하고 관리할 수 있도록 코드를 추가했다.
	: Menu 파일에서는 Line 도형을 선택할 수 있는 메뉴 항목을 추가했다.
	: main 파일에서는 Line 도형을 생성하고 처리하는 로직을 추가했다.

- 이 과정에서 어떤 점을 알 수 있었는가
	: 새로운 도형을 추가할 때, 관련된 여러 파일들을 꼼꼼히 확인하고 수정해야 한다는 점을 알 수 있었다.
	: 특히, 메뉴 선택 부분과 도형 관리 부분에서 새로운 도형에 대한 처리가 제대로 이루어져야 한다는 것을 배웠다.
	: 그래도 클래스를 활용한다면 새로운 도형을 추가하는 과정이 비교적 수월하고 쾌적하다는 점도 알게 되었다.
